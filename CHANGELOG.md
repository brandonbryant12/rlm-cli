# Changelog

## 0.5.0 — Initial release

- Full RLM-powered codebase analysis: security, architecture, review, documentation
- 5 custom REPL tools: grep_tree, list_files, file_stats, read_file, find_imports
- 5-layer config resolution (defaults → global → project → env → CLI)
- .gitignore-aware source tree loading with symlink escape protection
- Incremental refresh via git diff
- Debug and freeform Q&A commands
- Baseline caching with staleness detection
- Cost tracking and trace export
- Markdown and JSON output formats
